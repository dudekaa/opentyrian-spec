# Packaging Julius in RPM
[![Copr build status](https://copr.fedorainfracloud.org/coprs/nost23/opentyrian/package/opentyrian/status_image/last_build.png)](https://copr.fedorainfracloud.org/coprs/nost23/opentyrian/package/opentyrian/)

This project aims to create an **RPM** package for **opentyrian**, an open source re-implementation of Tyrian.

Opentyrian official repo: https://github.com/opentyrian/opentyrian

Build packages: https://copr.fedorainfracloud.org/coprs/nost23/opentyrian/

## Install
```
$ sudo dnf copr enable nost23/opentyrian
$ sudo dnf install opentyrian
$ opentyrian
```
